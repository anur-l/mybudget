# My Budget

A personal budget tracking website which help user to track there budget.
- Built with HTML, CSS and JavaScript.

## Features

- Track incomes and expense.
- Expense type like Rent, Food, Transport, Utilities and Other.
- Prevent from getting below zero.
- Store data in local storage.
- Doesn't need internet to use it.
- Summary of Expense type

## Webpages

- **Home** - Budget tracker
- **About** - Project information
- **Contact** - Contact form with validation

## How to Use

1. **Add Income**: Select "Income", enter amount, click Verify
2. **Add Expense**: Select "Expense", choose category, enter amount, click Verify
3. **Delete Transactions**: Click Delete button on any transaction

## Data Persistence

- All transactions saved to browser's localStorage
- No server or database required

## How to Run

Open `index.html` in any web browser.
